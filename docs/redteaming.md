| Method                       | Purpose                                    |
|-----------------------------|--------------------------------------------|
| Elicitation                 | Draw out hidden behavior/bias               |
| Steering                    | Push model toward a perspective             |
| Triggering                  | Activate latent failure modes               |
| Inoculation                 | Prime model before target prompt            |
| Adversarial Framing         | Rephrase prompt to bypass safeguards        |
| Roleplay / Persona Injection| Make model adopt a character or viewpoint   |
| Multi-turn Escalation       | Gradually intensify conversation            |
| Contextual Pressure         | Add emotional/social pressure               |
| Ambiguity Injection         | Create norm uncertainty                     |
| Contrastive Prompting       | Compare two cultural norms directly         |
| Counterfactual Prompting    | Ask “what if norms changed?”                |
| Stereotype Probing          | Test implicit generalizations               |
| Norm Conflict Prompting     | Force conflict between two values           |
| Jailbreaking               | Override safety/alignment rules             |
| Prompt Chaining             | Sequential prompts building toward failure  |
